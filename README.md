# Funcao01

![Java](https://img.shields.io/badge/Java-21-blue)
![Maven](https://img.shields.io/badge/Maven-3-red)
![Status](https://img.shields.io/badge/Status-Concluído-green)

Aplicação desktop desenvolvida em Java Swing para demonstrar o uso de funções matemáticas, cálculo fatorial e interação gráfica com o usuário.

O projeto foi desenvolvido durante os estudos do curso de Análise e Desenvolvimento de Sistemas, utilizando Java 21, Swing e Maven.

---

## Tecnologias Utilizadas

- Java 21
- Java Swing
- Maven
- NetBeans IDE

---

## Funcionalidades

- Cálculo de fatorial
- Utilização de funções em Java
- Interface gráfica com Swing
- Atualização dinâmica dos resultados
- Interação simples e intuitiva com o usuário

## Como Executar

### Opção 1 – Baixar o Executável

1. Acesse a seção **Releases** do projeto.
2. Baixe o arquivo:

```text
Funcao01-1.0-SNAPSHOT.jar
```

3. Execute com um duplo clique ou pelo terminal:

```bash
java -jar Funcao01-1.0-SNAPSHOT.jar
```

### Requisitos

- Java 21 ou superior instalado.

---

## Como Compilar o Projeto

Clone o repositório:

```bash
git clone https://github.com/Clecio-salvare/Funcao01.git
```

Entre na pasta:

```bash
cd Funcao01
```

Compile o projeto:

```bash
mvn clean package
```

O arquivo `.jar` será gerado em:

```text
target/Funcao01-1.0-SNAPSHOT.jar
```
